## ✨ 特性

- 🤖 **智能对话** - 支持多模型，工具调用，网页搜索
- 📔 **宠物日记** - 自动生成日记，记录生活点滴
- 🧠 **自我学习** - 记忆用户偏好，越用越懂你
- 🛠️ **技能系统** - 文件操作、命令执行、定时任务
- 🎮 **休闲娱乐** - 小游戏、商店、成就系统
- 🎬 **视频动画** - 透明窗口，多种状态动画

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/shipbetweenwm/GooseBaby.git
cd GooseBaby

# 安装依赖
flutter pub get

# 运行
flutter run -d windows
```
