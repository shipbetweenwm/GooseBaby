# 🦢 鹅宝 GooseBaby

> AI 驱动的桌面智能宠物伙伴

---

## 📖 产品简介

**鹅宝 (GooseBaby)** 是一款基于 Flutter 开发的 Windows 桌面宠物应用。它不仅是一个可爱的桌面伴侣，更是一个集成了大语言模型（LLM）的智能助手，能够与你自然对话、帮你完成任务、记录生活点滴。

鹅宝以一只呆萌的小鹅形象出现在你的桌面上，它会根据你的互动做出各种可爱的反应，同时具备强大的 AI 能力，成为你工作和生活中的贴心伙伴。

---

## ✨ 核心功能

### 🤖 智能对话

- **多模型支持**：支持通义千问、腾讯混元、OpenAI 等多种大语言模型
- **上下文记忆**：记住你们的对话历史，越聊越懂你
- **工具调用**：AI 可以直接调用各种技能帮你完成任务
- **网页搜索**：支持联网搜索获取实时信息
- **深度思考**：复杂问题时会进行深度分析

### 📔 宠物日记

- **自动记录**：每天晚上自动生成一篇日记，用鹅宝的口吻记录当天的互动
- **高光时刻**：自动捕捉并记录当天的精彩瞬间
- **心情追踪**：根据互动情况分析每日心情
- **重新生成**：不满意日记内容？一键重新生成

### 🧠 自我学习

- **记忆系统**：自动从对话中提取重要信息，学习你的偏好
- **用户画像**：记录你的姓名、职业、性格、爱好等信息
- **失败经验**：记住错误教训，避免重复犯错
- **分级记忆**：按重要性分级存储，高频经验永久记忆

### 🛠️ 技能系统

鹅宝内置丰富的技能，可以直接帮你处理各种任务：

| 技能 | 说明 |
|------|------|
| 📝 文件操作 | 创建、读取、编辑文件 |
| 💻 命令执行 | 运行 Shell 命令和脚本 |
| 🧠 记忆管理 | 查询和管理记忆内容 |
| ⏰ 定时任务 | 创建和管理定时提醒 |
| 🤔 深度思考 | 复杂问题的分析和推理 |
| 🔍 网页搜索 | 联网搜索实时信息 |

### 🎮 休闲娱乐

- **小游戏**：内置打地鼠、猜数字等小游戏
- **商店系统**：用互动积分兑换虚拟物品和装扮
- **成就系统**：记录里程碑，解锁成就徽章

### ⏰ 智能关怀

- **定时搭话**：主动找你聊天，不会让你感到孤单
- **健康提醒**：久坐提醒、喝水提醒、休息提醒
- **节日感知**：在特殊节日送上祝福
- **天气联动**：根据天气情况给出贴心建议

### 🎬 视频动画

- **透明窗口**：宠物透明显示在桌面上
- **多种状态**：开心、生气、睡觉、吃饭、玩耍等多种动画
- **随机切换**：空闲时随机播放无聊发呆、装萌等动画
- **交互反馈**：点击、拖拽时播放对应的动画反应

### 宠物主界面
- 透明窗口，宠物悬浮在桌面

- 显示统计信息和心情指数

### 设置面板
- AI 模型配置
- 技能管理（导入/删除）
- 定时任务设置
- 宠物偏好设置
- 工作目录配置

## 🏗️ 技术架构

### 前端技术
- **Flutter 3.16+**：跨平台 UI 框架
- **Provider**：状态管理
- **Hive**：本地数据存储
- **media_kit**：视频动画播放
- **window_manager**：桌面窗口控制

### AI 集成
- 支持 OpenAI 兼容 API
- 工具调用 (Function Calling)
- 多轮对话上下文管理
- Token 预算控制

### 技能系统
- 基于标准化的技能接口
- 支持 ZIP 包导入扩展技能
- Python/Shell 脚本执行
- 安全的沙箱环境

## 📦 安装使用

### 系统要求
- Windows 10/11 64位
- 最低 4GB 内存

### 快速开始

1. 下载发布版本或自行编译
2. 运行 `GooseBaby.exe`
3. 首次运行会提示配置 AI 模型
4. 获取 API Key 并填入设置
5. 开始与鹅宝互动！

### 自行编译

```bash
# 克隆项目
git clone https://github.com/shipbetweenwm/GooseBaby.git
cd GooseBaby

# 安装依赖
flutter pub get

# 运行
flutter run -d windows

# 编译发布版本
flutter build windows --release
```

## 🔧 配置说明

### AI 模型配置

在设置面板中配置：

| 模型 | API Key 来源 |
|------|-------------|
| 通义千问 | [阿里云 DashScope](https://dashscope.console.aliyun.com/) |
| 腾讯混元 | [腾讯云控制台](https://console.cloud.tencent.com/hunyuan) |
| OpenAI | [OpenAI Platform](https://platform.openai.com/) |

### 工作目录

在设置面板中可以自定义工作目录，AI 生成的文件会保存在该目录下。

## 📁 项目结构

```
GooseBaby/
├── lib/
│   ├── ai/              # AI 相关（LLM 集成、Prompt）

└── windows/             # Windows 平台配置
```
