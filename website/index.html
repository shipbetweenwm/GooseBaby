CUA (Computer Use Agent) — 它看得懂你的屏幕，能替你点鼠标、敲键盘、操作任何软件。

Security

让它干活，但别担心翻车

安全不是可选项，是默认行为。三层防护内置在每一次操作中，零配置开箱即用。

🔴

硬拦截

删库跑路？不存在的。rm -rf /、格式化磁盘等命令直接拒绝，连问都不问。

🟡

软提醒

sudo、chmod 777 这类有风险的操作，会先自己想一想，再问你确不确定。
