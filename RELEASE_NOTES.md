# 🦢 鹅宝 v1.0.0 - 首次发布

## ✨ 功能亮点

### 🎀 可爱的桌面宠物
- 11种生动动画表情（装萌、睡觉、工作、被撸等）
- 可拖拽、互动、双击触发动画

### 🧠 长期记忆陪伴
- 记住你说过的每一件事
- 理解上下文，越聊越懂你
- 主动关心和提醒

### 💬 智能对话
- 基于腾讯混元等 AI 大模型
- 温暖不油腻的陪伴感
- 支持多种 AI 模型切换

### 🤖 智能体技能系统
- 支持自定义技能包
- ZIP/文件夹导入
- 自动识别 SKILL.md 说明书

### 🎮 养成系统
- 属性管理（饥饿度、心情、清洁度、精力）
- 金币系统（在线挂机+对话获取）
- 成就系统

### 更多功能
- 📅 日记记录
- ⏰ 定时提醒
- 🛒 商店系统
- ⚙️ 丰富的设置选项

---

## 📥 安装说明

1. 下载 `GooseBaby-v1.0.0-Windows.zip`
2. 解压到任意目录
3. 双击 `goose_baby.exe` 启动
4. 首次使用需要配置 AI API Key（设置 → AI 模型）

---

## ⚙️ 系统要求

- Windows 10/11 (64位)
- 无需安装其他依赖
